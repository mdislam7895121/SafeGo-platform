# SAFEGO BD SHOP PARTNER – BANGLA-FIRST ULTRA-SIMPLE VERSION
# (Give highest priority to simplicity, Bangla language, and zero learning curve)

You must implement a Bangladesh-only Shop Partner experience that is:
1. As simple as water (“পানির মতো সহজ”)
2. Fully in Bangla language (UI+labels)
3. Requires no training or technical knowledge
4. Zero-complexity product listing
5. Supports full SafeGo rules without breaking anything
6. Designed for shopkeepers with minimal education

All rules from the SafeGo master ruleset MUST remain intact.
Do not modify or break any existing backend logic.

===================================================
A) BANGLA-FIRST UI (MANDATORY)
===================================================
All text for the Shop Partner dashboard MUST be in natural Bangla.

Examples:
- “দোকান সেটআপ”
- “পণ্য যোগ করুন”
- “দাম”
- “ছবি তুলুন”
- “সেভ”
- “অর্ডার দেখুন”
- “ডেলিভারি প্রস্তুত”
- “আপনার দোকান এখনো অনুমোদিত হয়নি”

No English except product names.

===================================================
B) ULTRA-SIMPLE ONBOARDING
===================================================
Create a 3-step “super simple” setup wizard that any non-technical Bangladeshi can complete:

### Step 1: দোকানের তথ্য
Fields:
- দোকানের নাম (shop_name)
- দোকানের ধরণ (dropdown with icons: মুদিখানা, মোবাইল দোকান, কসমেটিক্স, স্টেশনারি…)
- দোকানের ঠিকানা (shop_address)
- মানচিত্রে দোকান দেখান (tap-to-pin)
- দোকানের লোগো (optional: “ছবি তুলুন” button)

UI rules:
- Very large buttons
- Bangla instructions under each field
- Single page, big font

### Step 2: প্রয়োজনীয় সেটিংস
Fields:
- ঘরে ডেলিভারি চালু/বন্ধ (toggle)
- ডেলিভারি দূরত্ব (3km / 5km / 10km — big buttons)
- অর্ডার প্রস্তুতির সময় (dropdown: ১০/২০/৩০ মিনিট)

### Step 3: প্রথম পণ্য যোগ করুন (Super Simple Form)
Fields:
- পণ্যের নাম
- দাম (৳)
- ক্যাটাগরি (dropdown)
- ছবি (button: “ছবি তুলুন”)

Rules:
- Only 4 fields
- No extra complexity
- After first item → redirect to “আমার দোকান”

===================================================
C) ULTRA-SIMPLE PRODUCT SYSTEM
===================================================
Create a product management system so simple even a low-literacy person can use it.

### Product List Page
- Big button: “নতুন পণ্য যোগ করুন”
- List view:
  - product_photo (thumbnail)
  - product_name (Bangla-friendly)
  - price (৳)
  - Toggle: “চালু / বন্ধ”
  - Edit button: “সম্পাদনা”
  - Duplicate button: “কপি করুন”

### Add / Edit Product Form (Bangla)
Required:
- পণ্যের নাম
- দাম
- ক্যাটাগরি
- ছবি (single image)

Optional:
- বর্ণনা (short Bangla text)
- মাপ/ইউনিট (kg, piece, pack…)
- স্টক পরিমাণ

UX rules:
- ONE field per line
- Big font
- Bangla placeholder text
- Simple success message: “পণ্য সংরক্ষণ হয়েছে”
